The application is developed in Angular, Nodejs express framework and mongodb
Angular

1)THe folder name is frontend
2)run npm install to load the npm modules
3)ng serve to start the server
4)url : http://localhost:4200/

Nodejs:
1)run npm install to load modules
2)run npm index.js to start the backend server

MongoDB
1) please make sure the mongDB server is started with port 27017

After the all the three servers are started, open the url http://localhost:4200/ and go to browse and upload restrants button to upload the csv file.
